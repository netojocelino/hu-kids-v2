Sobre o aplicativo

O aplicativo HU Kids foi proposto, como trabalho pratico para a disciplina Engenharia de software II. Para o desenvolvimento uma alternativa aos jogos "free to play" disponíveis para dispositivos moveis.

O aplicativo e desenvolvido utilizando recursos gratuitos e software livre. Como as imagens disponíveis sob licença creative commons no site Open Game Art, e o motor gráfico Godot.

O aplicativo HU Kids também e uma forma de modernizar a brinquedoteca disponível para os pacientes da pediatria do Hospital Universitário de Sergipe, e ao mesmo tempo que diverte e distrai os jogadores o aplicativo também estimula hábitos saudáveis. Creative Commons.
